<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core MDX documentation generator with YAML frontmatter support
- Full component library including `<Callout>`, `<Steps>`, and `<Tabs>`
- Automatic escaping for `{`, `}`, `<`, and `>` characters

## Breaking Changes

- Removed H1 support; all pages now start with H2 headings
- Updated attribute syntax to require double quotes only

</Update>

## Upgrading to the Latest Version

Follow these steps to upgrade your Srishylam Jadi installation. You ensure smooth transitions by backing up your config first.

<Steps>
  <Step title="Backup Your Site" icon="database">
    Save your current documentation files and database.

    <Callout kind="tip">
      Use Git for version control: commit your changes before upgrading.
    </Callout>
  </Step>
  <Step title="Update Package" icon="download">
    Run the appropriate command for your package manager.
    <CodeGroup tabs="npm,yarn,pnpm">
      ```bash
      npm install srishylam-jadi@latest
      ```
      ```bash
      yarn add srishylam-jadi@latest
      ```
      ```bash
      pnpm add srishylam-jadi@latest
      ```
    </CodeGroup>
  </Step>
  <Step title="Rebuild and Deploy" icon="rocket">
    Rebuild your site and deploy to production.
    ```bash
    npm run build
    npm run deploy
    ```
  </Step>
</Steps>
